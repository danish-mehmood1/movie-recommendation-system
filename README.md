# 🎬 Movie Recommendation System

A Movie Recommendation System built using Collaborative Filtering with the MovieLens dataset, Railway MySQL database, and Streamlit.

## 🚀 Live Demo

https://movie-recommender-danish.streamlit.app

## ✨ Features

- Personalized movie recommendations
- Collaborative Filtering using Scikit-learn
- MySQL database hosted on Railway
- TMDB API integration for movie information
- Interactive Streamlit web interface

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- MySQL
- Railway
- TMDB API

## 📂 Dataset

MovieLens Latest Small Dataset

## ⚙️ Installation

```bash
git clone https://github.com/danish-mehmood1/movie-recommendation-system.git

cd movie-recommendation-system

pip install -r requirements.txt

streamlit run app.py
```
